## Flowchart Sistem Manajemen Data Mahasiswa

Dokumen ini berisi flowchart sederhana untuk sistem pengelolaan data mahasiswa, yang terdiri dari beberapa fitur utama seperti:

- **Tampilkan Menu**
- **Tambah Data**
- **Ubah Data**
- **Hapus Data**
- **Tampilkan Data**
- **Cari Data**
- **Kembali ke Menu**

Flowchart menggambarkan alur proses mulai dari *START*, pengguna memilih menu, melakukan aksi, lalu kembali ke menu hingga proses diakhiri (*END*).

## Isi Berkas
- Flowchart (gambar)
- README.md (penjelasan sistem)

## Cara Menggunakan
1. Buka file flowchart untuk memahami alur program.
2. Gunakan struktur tersebut jika ingin membuat aplikasi CRUD sederhana.
3. README ini memberikan deskripsi singkat untuk dokumentasi tugas atau proyek.
